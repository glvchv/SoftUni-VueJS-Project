# eventure

## Desription
```
SPA Web application for pre-ordering events' tickets (project for SoftUni VueJS-2020).

```
## Features

* Authentication
* Data validation
* CRUD operations


## Dependencies

* VueJS
* Vuelidate
* Vue-Router
* Vuetify
* Vuex
* Firebase


## Instructions

* npm install
* npm run serve
